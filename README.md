# SCT_ML_2
# Mall Customer Segmentation using K-Means Clustering

## Overview
This project applies the K-Means Clustering algorithm to segment mall customers based on their purchasing behavior. The model groups customers using features such as Age, Annual Income, and Spending Score.

Customer segmentation helps businesses understand customer patterns, improve marketing strategies, and make data-driven decisions.

---

## Objective
The objective of this project is to analyze customer data and divide customers into meaningful groups based on similar characteristics and spending habits.

---

## Key Features
- Data preprocessing and cleaning  
- Exploratory Data Analysis  
- Customer segmentation using K-Means Clustering  
- Cluster profiling  
- Data visualization  
- Business insight generation  

---

## Methodology

### 1. Data Collection
Loaded the mall customer dataset.

### 2. Data Preprocessing
Selected relevant numerical features for clustering.

### 3. Exploratory Analysis
Performed descriptive statistics and data understanding.

### 4. Model Training
Applied the K-Means Clustering algorithm to group customers.

### 5. Cluster Profiling
Calculated average values for each cluster.
